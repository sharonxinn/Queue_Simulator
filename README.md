# Project Description
Developed a queue simulation system to model vehicle arrivals and refueling operations at a petrol station with four fueling pumps and two dedicated queuing lanes, following real-world operational constraints.

# Skill Performed
-Designed a petrol station queue simulation with 2 lanes and 4 pumps
-Implemented random number generation (LCG / rand) for simulation inputs
-Simulated vehicle inter-arrival time, petrol type, refueling time, and quantity
-Created probability distribution tables (Probability, CDF, random number range)
-Simulated real-time events (arrival, queueing, refueling start/end)
-Applied queueing rules to assign vehicles to correct lanes and pumps

# Key Insights
-Simulated both non-peak and peak hour scenarios
-Generated tables for:
Arrival & inter-arrival time
Lane/queue assignment
Refueling start & end time
Waiting time & total time in system
-Calculated:
Average waiting time
Average time in system
Probability of waiting
Average service time per pump
-Compared system performance (peak vs non-peak)
